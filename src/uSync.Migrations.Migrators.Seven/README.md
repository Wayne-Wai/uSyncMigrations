## Legacy Legacy Migrators. 

At the moment this version of uSync.Migrations doesn't support migrating all the way 
from umbraco 7 to umbraco 17, 

This library is a placeholder (and doesn't form part of the package) for these migrators
should we in the future get our finger out and write something to convert uSync v7 files
to the v8+ format. 

if we do that then these migratiors will 'just' work. and we can merge them back in.

however - at this time they are UNTESTED, so we can't say they will work if/until we
get to that point. 

if you are coming from v7 - using uSync.Migrations for Umbraco v13, to get to a modern
version of the files.
